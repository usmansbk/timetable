# Timetable

Timetable app for Students

## Features

- [x] Offline support

- [x] Repeating events

- [x] Infinite scroll agenda

- [x] Event reminder notification

- [x] Dark theme

## Getting Started

### Prerequisites

- NodeJS v16

- Yarn

### Setup

```sh
git clone https://github.com/usmansbk/timetable.git
cd ./timetable
```

### Install

```sh
yarn
```

### Run

```sh
yarn android
```

```sh
yarn ios
```
