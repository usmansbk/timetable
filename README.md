## Getting Started

### Prerequisites

- [NodeJS](https://nodejs.org/en/) v16

- [Yarn](https://yarnpkg.com/getting-started/install)

### Setup

```sh
git clone https://github.com/usmansbk/timetable.git
cd ./timetable
```

### Install

```sh
yarn
```

### Run

```sh
yarn android
```

```sh
yarn ios
```
